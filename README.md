# 🌾 Edau Farm - West Pokot's Premier Sustainable Farm

West Pokot's premier sustainable agricultural enterprise. Premium Acacia honey, fresh seasonal fruits, Dorper sheep, and free-range poultry — all raised with traditional wisdom and modern sustainable practices.

## 🏡 About Edau Farm

Founded in 2015, Edau Farm has been committed to sustainable agriculture that honors our ancestral lands while embracing eco-friendly farming methods. Our farm produces:

- **Premium Acacia Honey** - Raw, unprocessed honey from the acacia forests of West Pokot
- **Fresh Seasonal Fruits** - Mangoes, pawpaws, passion fruits, and avocados
- **Dorper Sheep** - Quality breeding stock and meat goats raised on natural pastures  
- **Free-Range Poultry** - Healthy chickens and eggs from pasture-raised hens
- **Fresh Vegetables** - Organic vegetables from our gardens

## 🛠️ Technology Stack

- **Frontend**: Next.js 14 (App Router), React 18, TypeScript
- **Styling**: Tailwind CSS
- **Database**: MongoDB + Mongoose ODM
- **State Management**: Zustand
- **Payments**: M-Pesa (Lipia)
- **Images**: Cloudinary
- **Email**: Resend

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📦 Project Structure

```
edau-farm/
├── app/              # Next.js App Router pages
│   ├── api/          # API routes
│   ├── products/     # Product pages
│   ├── cart/         # Shopping cart
│   ├── checkout/     # Checkout flow
│   └── admin/        # Admin dashboard
├── components/       # Reusable UI components  
├── lib/             # Utilities and MongoDB client

├── store/           # Zustand stores
└── public/          # Static assets
```

## 🛍️ Features

- **Product Catalog**: Browse honey, fruits, livestock, poultry, and vegetables
- **Shopping Cart**: Add items from the farm with quantity management
- **Secure Checkout**: M-Pesa payment integration
- **User Accounts**: Sign up and manage orders
- **Admin Dashboard**: Manage products, orders, and customers
- **Responsive Design**: Works on mobile and desktop
